<p align="center">Create a new simple project in HTML5 with one command.</p>

<br/> 

### Install
```
npm i -g create-html5-app
```

OR:

### Usage

```
npx create-html5-app app-name
```

```
cd app-name
npm run dev
#or
yarn run dev
```
